HTML Images Syntax

The HTML tag is used to embed an image in a web page.

Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image. The tag is empty, it contains attributes only, and does not have a closing tag. The tag has two required attributes:

    • src - Specifies the path to the image
      alt - Specifies an alternate text for the image
